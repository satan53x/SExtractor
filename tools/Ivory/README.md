## Ivory Tools
由`Cosetto`提供
* 解包: python pk.py -u <archive_base> <output_folder>
* 封包: python pk.py -p <input_folder> <output_archive>
* 剧本解码: python ocb.py -d <input.ocb> <output_dir>
* 剧本编码: python ocb.py -c <input_dir> <output.ocb>
* 图片解码: python sg.py -d <input_sg> <output_png>
* 图片编码: python sg.py -e <input_png> <output_sg>
* 测试游戏: `ネトッテヤ`
