### dat_decrypt
由`Steins;Gate`提供，针对特殊版本的NScripter
* 解密/加密: python dat_decrypt.py <input_file> <output_file>
* 需要自行进行一次xor `0x84`，可以使用`tools/_common/xor_file.py`
* 需要在脚本里设置密钥（Garbro有部分预设），寻找密钥的教程（包含全流程）：[nscripter教程](https://www.ai2.moe/topic/46511-%E7%89%B9%E6%AE%8A%E5%8A%A0%E5%AF%86%E7%89%88%E6%9C%AC%E7%9A%84nscripter%E5%BC%95%E6%93%8E%E5%AF%BB%E6%89%BE%E5%AF%86%E9%92%A5%E6%95%99%E7%A8%8B)
